HTML & CSS QUICK-REFERENCE GUIDE

HTML Guide:

HTML (HyperText Markup Language) is the standard markup language for documents designed to be displayed in a web browser.

HTML Tags:

Text Tags

  • <h1>-<h6> - Header tags 
    <h1>Main Heading</h1>
  • <p> - Paragraph tag 
    <p>This is a paragraph.</p>
  • <br> - Line break 
    This is a line.<br>This is another line.

Anchor & Media Tags

  • <a href> - Link tag 
    <a href="https://example.com">Visit Example</a>
  • <img src> - Image tag 
    <img src="image.jpg" alt="Example" width="200" height="150">

List Tags

  • <ol> - Ordered list 
    <ol>
  <li>Item 1</li>
  <li>Item 2</li>
</ol>
  • <ul> - Unordered list 
    <ul>
  <li>Item A</li>
  <li>Item B</li>
</ul>
  • <dl> - Description list 
    <dl>
  <dt>Term 1</dt>
  <dd>Definition 1</dd>
  <dt>Term 2</dt>
  <dd>Definition 2</dd>
</dl>

Table Tags

  • <table> - Table structure 
    <table border="1">
  <tr>
    <th>Header 1</th>
    <th>Header 2</th>
  </tr>
  <tr>
    <td>Data 1</td>
    <td>Data 2</td>
  </tr>
</table>

Comment Tag

<!-- Comment --> - Used to add comments in HTML code. 
<!-- This is a comment -->

CSS Guide

CSS (Cascading StyleSheets) is used to style and design webpages, controlling elements like color, font, size, spacing, and positioning.

CSS Properties:

  • color - Sets text color 
    p { color: blue; }
  • background-color - Sets background color 
    body { background-color: lightgray; }
  • font-size - Sets text size 
    h1 { font-size: 24px; }
  • margin - Sets outer spacing 
    div { margin: 10px; }
  • padding - Sets inner spacing 
    div { padding: 15px; }
  • border - Defines a border 
    table { border: 1px solid black; }
  • display - Defines layout 
    div { display: flex; }
  • text-align - Aligns text 
    h1 { text-align: center; }
